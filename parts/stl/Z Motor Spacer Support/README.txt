# Summary

**24/05/2018: Updated to v2 which reduces filament by about 25% and print time by about 15%.**

This will support the Z-motor and align it to prevent the leadscrew from binding. You will no longer have to use paper/card or leave the motor loose to prevent the binding. 

Check pics for installation.

**Rotate the model in Cura like I've shown in the pic before printing.**
